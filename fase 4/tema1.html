FUNDAMENTOS DE HTML

En el documento de (Celaya,2014) se puede expresar que un archivo HTML es un archivo de texto solamente, es decir que con el editor de texto de Windows se puede modificar y crear un archivo HTML. Para poder crear un archivo o documento HTML se debe realizar el paso a paso:

  1. Se debe dar clic en el botón dereche del equipo,
  2. Luego le vamos a dar clic en nuevo,
  3. Después de eso vamos a darle en nuevo documento de texto,
  4. Seguidamente de que hallamos creado un archivo de texto nuevo, procedemos a renombrar nuestro archivo con la extensión .HTML,
  5. Finalmente, le damos clic en SÍ en que estamos seguros de cambiarla.
Seguido a ello, después de que este creado nuestro documento procedemos a:
  1. Abrir el bloc de notas (la App),
  2. Este se encuentra en archivo,
  3. Después de desplegar el módulo archivo pulsamos o le damos clic en abrir,
  4. Luego de seleccionar ello, procederemos a darle clic a nuestro archivo creado en la parte inferior de la ventana donde aparecerá seleccionar todos los archivos,
  5. Y finalmente abrimos o le damos clica nuestro archivo para comenzar a escribir nuestro código.
Se puede decir que el lenguaje para nuestro código, es un lenguaje de atiquetas como algunos lo han mencionado; se dice que las herramientas o palabras especiales que se utilizan son denominadas etiquetas debido a que se debe conformar para la elaboración de una página web. Aunque se puede decir que existen algunas excepciones, en la cual se utilizan en pares. Todos los documentos de HTML tienen los mismos elementos, tales como: Una estructura que contiene etiquetas como Doctype que es la primera línea de código que encontramos; Head que delimitan el contenido de la cabecera del documento; Title para agregar un título en el navegador; Body que rodean el contenido visual de la página creada. Dentro de las etiquetas más comunes podemos encontrar: !--comentarios-- que se utiliza para escribir las anotaciones importantes que tengamos para nuestro código, este no se ve reflejado en el navegador; para poder formar textos tenemos b o /b para aplicar negrita dentro del texto que se encuentran dentro de las etiquetas; tenemos la i o /i para aplicar cursiva al texto que se encuentra dentro de las etiquetas; u o /u para aplicar cursiva; font o /font para delimitar un texto con un formato de fuente que está defeinido por sus propiedades, dentre de este podemos encontrar font sice=x o /font para indicar el temaño del texto, font color="#RRGGBB" que ayuda a definir el color del texto, font face "arial"; la etiqueta h1, h2, h3, h4, h5, h6 se aplica para indicar el temaño de la letra (indicando que h1 es mayor). Para la creación del formato de los párrafos encontramos a br que es usado para introducir saltos de línea; p o /p que se utiliza para delimitar un párrafo, dentro de ellos encontramos p align='center' se usa para centrar el texto, p align='left' para que el texto esté alineado a la izquierda, p align='right' para que el texto esté alineado a la derecha; p align='justify' se usa para que el texto del párrafo sea justificado; hr se usa para insertar una línea de forma horizontal; div se usa para delimitar una sección del documento. En la creación de lista podemos ver una la lista no numerada que es con la etiqueta ul y para colocar varios renglones agregaremos la etiqueta li en el siguiente renglón; tamién podemos encontrar la lista numerada que se crea con la etiqueta ol y hacemos lo mismo si queremos colocarle más renglones, usamos la etiqueta li x el número de renglones que se necesiten, al igual que la lista no numerada se cierra automaticamente com la etiqueta /ol. Tambén están las etiquetas para subir las imágenes, la cual es: img src="dirección de la imagen" que es utilizada para insertar una imagen en dicha ruta "dirección de la imagen"; para la creación de tablas donde agregamos la etiqueta table o /table en donde se define el comienzo y final de una tabla, tr o /tr que se aplica para indicar el comienzo y el final de cada una de las filas de la tabla, también encontramos td o /td que nos indica el inicio y fin de cada una de las columnas o celdas dentro de las filas; en la creación de enlaces podemos agregar a href="http://www.ejemplo.com" para una página web y para colocar una dirección de correo se coloca href="mailto:ejemplo@ejemplo"; para agregar formularios en nuestra página web colocaremos form o /form para marcar el inicio y el final de la creación de un formulario.
